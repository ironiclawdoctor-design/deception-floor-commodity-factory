================================================================================
CONFIGURATION OPTIONS
================================================================================

Command-line flags:
  --task DESCRIPTION          Required: Task description for classification
  --prompt PROMPT             Required: The actual prompt/request text
  --model MODEL               Optional: Force specific model (bitnet|haiku)
  --cost-estimate COST        Optional: Override default cost estimate

Examples:
  ./tier-routing-enforcement.sh --task "What is 2+2?" --prompt "2+2"
  ./tier-routing-enforcement.sh --task "Write a story" --prompt "..." --model haiku
  ./tier-routing-enforcement.sh --task "Bash code" --prompt "..." --cost-estimate 0.00150

Adjustable parameters (in script):
  BITNET_HOST = "127.0.0.1"
  BITNET_PORT = "8080"
  BITNET_TIMEOUT = 2 seconds
  Default cost (BitNet) = $0.0000
  Default cost (Haiku) = $0.00081

SUPPORT & CUSTOMIZATION
================================================================================

Add custom classification:
  Edit BITNET_PATTERNS or HAIKU_PATTERNS in the script
  Add new regex patterns as needed
  Restart: no service restart required

Override for specific tasks:
  Use --model flag to force routing
  Use --cost-estimate to override cost

  Modify call_haiku() to exit with error
  Strict routing: BitNet only (no external calls)
