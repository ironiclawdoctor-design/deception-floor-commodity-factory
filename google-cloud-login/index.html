🔧 Manual Cookie Export Immediate

Export cookies directly from Chrome DevTools. Works now, no setup required.

  1. Open Chrome and navigate to console.cloud.google.com
  2. Log in with your Google account (if not already)
  3. Open DevTools (F12) → Application tab → Storage → Cookies
  4. Select cookies for:
    • .google.com
    • console.cloud.google.com
    • accounts.google.com
  5. Right-click → "Export cookies as JSON"
  6. Save file as google-cookies.json in workspace secrets
# Save to:
/root/.openclaw/workspace/secrets/google-cookies.json

🔐 OAuth 2.0 Flow Recommended

Use official Google OAuth to get access & refresh tokens. Programmatic, headless.

  1. Create Google Cloud project (claim $300 credits)
  2. Enable APIs: Cloud Run, Functions, Firestore, Scheduler
  3. Create OAuth 2.0 credentials (Desktop app type)
  4. Download client_secret.json
  5. Place at /root/client_secret.json
  6. Run the authentication script:
cd /root

Result: Tokens stored in ~/.config/gog/, ready for GCP CLI.

🤖 Service Account (Headless) Most Secure

Create a service account with limited permissions. No browser, no session expiry.

  1. In Google Cloud Console: IAM & Admin → Service Accounts
  2. Create new service account with roles:
    • Cloud Run Admin
    • Cloud Functions Developer
    • Service Account User
  3. Generate JSON key (do NOT share publicly)
  4. Download key file
  5. Upload to OpenClaw secrets directory:
/root/.openclaw/workspace/secrets/gcp-service-account.json

Security: This key allows programmatic access to GCP resources. Store securely.
