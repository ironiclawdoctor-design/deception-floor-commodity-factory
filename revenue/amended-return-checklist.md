## Checklist — For Each Year

### 1. Home Office Deduction (Form 8829)

- [ ] Did you use any part of your home exclusively and regularly for business?
- [ ] Even a corner desk counts if it's dedicated to business use
- **Method:** Simplified — $5/sq ft, max 300 sq ft = max **$1,500/year**
- If yes: this deduction is available for EVERY year you worked from home

**Estimated value per year:** $250–$1,500 (depends on sq footage)

### 2. Self-Employment / Side Business Expenses

- [ ] Did you have ANY freelance, consulting, gig, or side income in 2022/2023/2024?
- [ ] If yes: were ANY of these expenses deducted on the original return?
  - Software subscriptions
  - API / cloud service costs
  - Phone (business %)
  - Internet (business %)
  - Professional development / courses

**If you had side income and didn't file Schedule C:** This is the biggest miss.

### 3. Startup Costs — Section 195

- [ ] Did any business activity begin in 2022, 2023, or 2024?
- If yes: up to **$5,000** of startup costs deductible in the year the business began
- Qualifying costs: research time, software trials, domain registration, planning, courses
- **This is a one-time deduction in the first year — if missed, it's worth amending for**

### 4. Qualified Business Income (QBI) — Section 199A

- [ ] Did you have net SE income in any of these years?
- If yes: automatic **20% deduction** on that net income — no extra forms needed
- Many preparers miss this for small side businesses

### 5. Self-Employed Health Insurance

- [ ] Did you pay any health insurance premiums not fully covered by your W-2 employer?
- If yes: **100% deductible above-the-line** if you had net SE income

### 6. Self-Employment Tax Deduction

- If you paid SE tax (15.3%) on any side income: **50% of it is deductible** (Schedule 1, Line 15)
- Most basic tax software does this automatically, but check the original return

## How to File a 1040-X

1. **Get your original return:** IRS.gov → "Get Your Tax Record" → "Get Transcript" → Tax Return Transcript
2. **Identify all missed deductions** using this checklist
3. **Recalculate** net income → SE tax → SE deduction → QBI → AGI
4. **Complete Form 1040-X:** Column A = original amounts, Column B = net change, Column C = corrected amounts
5. **Attach amended Schedule C** (and Form 8829 if home office)
6. **Mail or e-file** (e-file available for 2020+ returns via IRS Free File or tax software)
7. **Wait 8–16 weeks** for refund check or direct deposit
