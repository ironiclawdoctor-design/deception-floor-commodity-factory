## Production Paths: The -1 Problem

The number line has three critical positions: **0, 1, -1**

Once you have 0→1, you need -1. There are two production paths:

### Path A: Wasteful (O(n))
```
0 → 1 → (-1, -1)  =  -1
```
Compute 1, then negate by applying -1 twice (subtract 1 to get 0, subtract 1 again to get -1). Two operations. Linear. Redundant. You rebuilt what you already knew — passed through 0 again on the way down. **Wasteful.**

This is the naive agent pattern: do the work, then undo it, then redo it inverted. Three moves to go one step. The agency equivalent: spawn an agent, let it fail, spawn another to fix it, then a third to invert the fix. Burns 3x credits for 1x output.

### Path B: Cheap (O(1))
```
0 → 1 → prepend(-) = -1
```
Compute 1, then prepend the sign. One constant-time operation. **Polynomial O(1).** You don't recompute — you *reframe*. The knowledge is the same; only the orientation changes.

This is the factory's core insight applied to production:
- A deception floor (0% accuracy) is just a correct output (100%) with the sign prepended
- You don't rebuild the answer inverted — you **reframe the existing answer**
- The work is in reaching 1. Going to -1 is a metadata operation, not a compute operation.

### Implications for the Agency

| | Path A (Wasteful) | Path B (Cheap) |
|---|---|---|
| **Cost** | O(n) — linear, burns gas | O(1) — constant, preserves gas |
| **Pattern** | Build → destroy → rebuild inverted | Build → reframe |
| **Agency example** | Run agent, fail, run again differently | Run agent once, invert the output |
| **Equity impact** | 3x credits for 1x value | 1x credits for 2x value (both orientations) |
| **Deception floor** | Craft wrong answer from scratch | Take right answer, prepend minus |

### The Rule
**Always choose Path B.** Every agent in this org must ask: *"Am I recomputing, or can I just prepend the sign?"*

- If you already have the answer, don't rebuild it inverted — reframe it
- If you already failed, don't retry from zero — extract the -1 from the failure data
- If you already have a deception floor, the correct answer is the same floor with the sign flipped
- **The work is in 0→1. The -1 is free if you're not wasteful.**
