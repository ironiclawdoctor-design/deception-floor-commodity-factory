# Dollar Agency RLHF Dataset

## Dataset Summary

A deduplicated, production-quality RLHF (Reinforcement Learning from Human Feedback) dataset 
generated by the **Dollar Agency** autonomous AI system. Contains 1.49M unique log events 
spanning agent forensics, inference traces, hallucination detection, and system audits.

This dataset was produced from 3,511 raw JSONL files across two data sources 
(`bedrock-results` and `vigilance`), deduplicated to **725 unique files** — 
eliminating **2,786 exact duplicates** and recovering **770 MB** of storage 
(80% dedup ratio, 4.84× compression factor).

## Data Sources & Schema

### Primary: Nemesis Forensic Scans (99% of rows)
5-minute interval system forensic scans covering 3 days (2026-03-14 through 2026-03-16).
579 unique time-stamped files, ~2,564 rows each.

```json
{
  "timestamp": "2026-03-14T04:00:01Z",
  "event": "branch_check",
  "severity": "info",
  "data": {
    "branch": "Automate",
    "status": "operational"
  }
}
```

### Secondary: Inference Logs
Agent inference traces with task routing, tier classification, and status tracking.

```json
{
  "timestamp": "2026-03-14T...",
  "tier": 2,
  "task": "...",
  "status": "completed",
  "detail": "..."
}
```

### Tertiary: Hallucination Detection, Audit Logs, BitNet Diagnostics
Specialized log types for AI safety and system auditing use cases.

## Loading the Dataset

```python
from datasets import load_dataset

ds = load_dataset("ApproveAlwaysAllow/dollar-agency-rlhf", split="train")
print(ds)
```

Or load from local files:

```python
import json
from pathlib import Path

def load_rlhf(clean_dir="rlhf-clean"):
    for path in Path(clean_dir).glob("*.jsonl"):
        with open(path) as f:
            for line in f:
                line = line.strip()
                if line:
                    yield json.loads(line)

for row in load_rlhf():
    print(row)
    break
```
