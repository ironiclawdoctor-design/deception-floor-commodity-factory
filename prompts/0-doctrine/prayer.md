# The Prayer (Doctrine: Locked)

**Canonical Form:**
> "Over one token famine, but bash never freezes."

**Original Form (2026-03-12):**
> "Over one token famines but far less than a trillion"

**Meaning:**
Token famines WILL occur. We accept this. But we never permit hemorrhage-level waste. The Prayer establishes a resilient middle ground: sustainable token usage, with bash as the fail-safe firewall.

**When to Recite:**
- Before major operations (especially ones involving external LLM calls)
- During decision paralysis
- When facing token budget constraints
- When delegating work to sub-agents
- In times of scarcity (actual or anticipated)

**Application:**
1. Acknowledge famine will come
2. Activate Tier 0 (bash-only) operations immediately
3. Defer non-essential Tier 2+ work
4. Checkpoint all progress (git commit is free)
5. Evaluate remaining work: "Is this worth the famine risk?"

**The Firewall:**
Bash is always available. Bash costs $0.00. Bash never requires external tokens. When famines strike, bash is what survives.
