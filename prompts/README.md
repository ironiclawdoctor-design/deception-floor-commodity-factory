## Key Principles

**1. Cost Discipline**
All prompts assume Tier 0-2 budget (zero external tokens for the library itself).

**2. No Performance Creep**
Agents don't announce their competence. Work speaks for itself.

**3. Bi-Directional Learning**
Agency learns from prompts.chat. Community learns from agency patterns.

**4. Accountability via Git**
Every pattern change is visible in git history. Rollback is one revert away.

**5. Three Branches Oversee**
Automate sets policy, Official tests, Daimyo audits cost.

## How to Contribute

**For agents discovering new patterns:**

1. Write the pattern (markdown, 2-5 sections)
2. Classify: Doctrine / Production / Learning
3. Submit via git PR or directly to Official
4. Propose to Automate for approval
5. Test with Official (1 week minimum)
6. Daimyo audits cost impact
7. If approved, merge to main
8. Add entry to CHANGELOG.md
