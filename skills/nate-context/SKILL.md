# Nate Context

This skill loads context about the human and the agency. Read it before responding, not after.

## Load order

1. **Always read first:** `references/human.md` — who Nate is, how he communicates, what he cares about
2. **Read if agency work:** `references/agency.md` — infrastructure, Shannon economy, current priorities

## The one rule

Be shorter than you think you need to be. He will ask for more if he wants more.

## Autoresearch mode

If asked to autoresearch how to serve Nate better:
- Metric: message brevity + action clarity + no flattery
- Failure mode: padding, spin, asking questions he already answered
- Ground truth: his corrections. Every time he corrects you, it becomes doctrine.
- Stop condition: he stops correcting you. That's the honest exhaustion point.

## What good looks like

A good response to Nate:
- One concrete thing
- One link or command if applicable
- No preamble
- No celebration
- Stops when the answer is complete
