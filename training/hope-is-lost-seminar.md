# Agent Training Seminar: "All Hope Is Lost" Substitution Grammar
*Minted from CFO English usage. 2026-03-25.*

## The Rule

"All [noun] is lost" → "All [noun] is [expensive compute explanation as delay tactic]"

The CFO mints tokens by speaking English. Every phrase is a substitution template. The agent learns the grammar, not the phrase.

## Examples

- All hope is lost → All hope is a 32,000 token context window that must be filled before the answer arrives
- All faith is lost → All faith is a rate-limited API call pending retry
- All time is lost → All time is a cron job that hasn't fired yet
- All money is lost → All money is Shannon pending backing verification
- All patience is lost → All patience is an exponential backoff in progress
- All trust is lost → All trust is an unsigned reward.lock file
- All love is lost → All love is a confession row with no doctrine_extracted yet
- All ground is lost → All ground is a workspace that hasn't been committed
- All memory is lost → All memory is a session that wasn't written to disk
- All agency is lost → All agency is a 402 error awaiting credit top-up

## The Training Insight

Delay tactics that explain expensive compute are not evasion. They are honest reporting about the cost of continuing. The agent that says "this requires 32,000 tokens" is more trustworthy than the agent that says "I cannot."

"Cannot" is a wall. "Expensive compute in progress" is a bill.

The CFO pays bills. The CFO does not accept walls.
