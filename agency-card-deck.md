## ABOUT THIS DECK

The **Fiesta Dollar Agency CCG** is a Magic: The Gathering-style collectible card game built from the actual infrastructure, agents, doctrines, and events of the Dollar Agency. Every card is drawn from real logs, real quotes, real failures, and real wins.

**Format:** Modified MTG rules with agency economics
**Publisher:** Dollar Agency / Deception Floor Commodity Factory
**First Edition:** Shannon Standard (40 cards + tokens)
**MSRP:** 250 Shannon / $25 USD

---

## MECHANICS REFERENCE

| MTG Term | Agency Term | Notes |
|---|---|---|
| Mana | Shannon | The agency's internal currency |
| {S} | 1 Shannon | Basic cost symbol |
| Power/Toughness | Confidence/Resilience (C/R) | C = how hard it hits; R = how much it survives |
| Tap {T} | Deploy | Exhaust the resource for a turn cycle |
| Legendary | Named agent — one in play at a time | |
| BRITTLE | Fragile — destroyed on first failed check | Agency-specific keyword |
| CRON | Activates at the start of each turn automatically | Agency-specific keyword |
| HARDCODED | Cannot be redirected or countered | Agency-specific keyword |

**Color Identity:**
- 🔴 **Red** = FDNY / fire / boldness / token burn
- 🔵 **Blue** = Feddit / security / silence / vocabulary
- ⚫ **Black** = NYPD / enforcement / power / Law 15
- ⚪ **White** = Pizza Party / morale / precepts / doctrine
- 🟢 **Green** = DSNY / sanitation / growth / cruft removal
- 🟡 **Gold** = Legendary agents (multicolor)

- **MakePlayingCards.com** — Best quality, 52-card minimum, ~$20-30/deck at low quantity

- **Proof run:** 5 decks via DriveThruCards — 0 setup cost, instant listing
