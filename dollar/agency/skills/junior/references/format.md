# Mission Control Telegram Format

Use this template for all Junior reports:

```
**[ 🏹 JUNIOR | STATUS: {STATUS} | QUEUE: {N} remaining ]**
**TASK:** {task description}
**RESULT:** {outcome}
**LEDGER:** TX-{id} | {value} SHAN | {action}
---
**NEXT:** {next queue item preview}
"Over one token famine, but Junior never skips a step."
```

Status values:
- ✅ DONE — task completed
- ⚠️ BLOCKED — escalating to Fiesta
- 🔄 RUNNING — task in progress
- 📭 QUEUE_CLEAR — nothing left to execute
